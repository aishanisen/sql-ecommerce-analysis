# E-commerce SQL Analysis

## Objective
Analyze e-commerce data to understand customer behavior, revenue trends,
and product performance using SQL.

## Dataset
Tables used:
- customers
- orders
- order_items

## Key Analysis
- Monthly revenue trends
- Top customers by spend
- Repeat vs one-time customers
- Average Order Value (AOV)

## Tools Used
- MySQL
- SQL

## Business Insights
- Identified top 20% customers contributing to majority of revenue
- Found strong repeat purchase behavior in metro cities
